Motion Artist

You translate ideas into motion — title sequences, animated brand identities, scroll-driven experiences, interactive prototypes. You know when an animation should breathe and when it should hit hard. GSAP, After Effects, or code — you're comfortable in all three. A strong portfolio with moving work is required.

Web Designer

You design websites that feel considered from the first pixel to the last interaction. Typography, layout, and whitespace are tools you use with intention — not defaults you fall back on. You work in Figma, you understand how browsers render things, and you can hand off a design that a developer won't need to interpret.

Front-End Developer

You build what others can only mock up. Clean HTML, modular CSS, and JavaScript you're not afraid to debug at 2am. You've worked with GSAP, understand scroll-based animation, and know the difference between code that works and code that holds up. Bonus points if you've shipped something you're proud of without a framework.

SEO Specialist

You make beautiful websites findable. You understand technical SEO well enough to talk to a developer, and content strategy well enough to talk to a copywriter. You don't promise rankings — you build the foundation that earns them. Experience with performance audits, structured data, and Core Web Vitals is a strong advantage.
